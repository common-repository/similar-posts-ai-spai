== Description ==

= Creates an AI-based recommended articles widget. The fastest plugin, since all calculations take place on the developer's servers. =

Fastest related Posts Plugin for WP

== Templates ==
5 design templates are available

== Statistic ==
Convenient and useful statistics inside the admin panel (screenshot-4.png)

== Monetization ==
You can monetize your traffic. The plugin will show native ads, and the money will be credited to your account. Money withdrawal works on all popular payment systems

== Installation ==

1. Upload `SPAI` plugin to the `/wp-content/plugins/` directory or add through wp admin panel
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Register or log in on developer's website. After adding the site, you will be given an API key.
1. Enter the API key in the plugin settings
